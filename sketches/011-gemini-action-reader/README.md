## Variant: Gemini Action Reader

### Design stance

保持 `gemini/3.html` 的左侧列表 + 右侧文档，但把 first 30 minutes 提升为右侧 sticky action panel。

### Key changes from gemini/3.html

阅读正文更宽松，行动区独立固定，强调“读完马上开工”。左侧还有 Read/Build 模式切换的雏形。

### Interaction

切换 idea、切换 Read/Build 模式、复制 first action。

### Best for

如果你喜欢 Gemini 3 的质感，同时希望产品更强地推动用户立刻执行。
