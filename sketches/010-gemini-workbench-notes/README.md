## Variant: Gemini Workbench Notes

### Design stance

在 `gemini/3.html` 的两栏基础上增加第三栏 notes，让它更像长期使用的个人项目资料库。

### Key changes from gemini/3.html

左侧是分组资料库，中间是文档详情，右侧是用户自己的选择理由和 checklist。适合“比较方案、沉淀判断”。

### Interaction

筛选分组、切换 idea、编辑 memo、勾选 checklist。

### Best for

如果产品要从一次性推荐变成长期 personal workbench。
