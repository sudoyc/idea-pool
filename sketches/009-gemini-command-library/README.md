## Variant: Gemini Command Library

### Design stance

保留 `gemini/3.html` 的 split-pane 和高级暗色气质，但把左上输入强化成 Raycast-ish command brief。

### Key changes from gemini/3.html

左侧不再像普通表单，而是一个当前生成上下文的命令块；下方仍然是 library list，右侧继续是文档阅读。

### Interaction

点击 quick brief 改写当前上下文；点击列表切换右侧文档；Copy/Start 给 toast 反馈。

### Best for

如果你喜欢 Gemini 3 的风格，但希望输入区更有“开发者工具”的启动感。
