## Variant: Raycast Command Nebula

### Design stance
把灵感池变成一个命令面板：输入即筛选，项目候选像命令一样可执行。

### Key choices
- Layout: 大型居中应用窗口，主命令列表 + 右详情。
- Typography: 大标题清晰，列表项像 launcher command。
- Color: 深紫黑背景，紫色发光，macOS/Raycast 窗口感。
- Interaction: 点击命令切换详情，搜索和按钮有 toast 反馈。

### Trade-offs
- Strong at: 启动器感强、交互明确、容易做键盘优先。
- Weak at: 内容密度比 Linear 版低，长文本展示弱。

### Best for
- 想把产品定位成“项目启动器 / command center”的方向。
