## Variant: Interactive Notion Block

### Design stance

Writing surface. The generator reads like an editable note rather than an app form followed by results.

### Key layout choices

Narrow centered document, inline sentence inputs, idea tabs as document blocks, h2-driven output sections, and a checkbox first action.

### Typography choices

23-30px opening sentence, 32-46px document title, 16px body copy with 1.82 line height, and minimal monospace for tags/metadata.

### Interactions

Inline prompt fields are contenteditable, tabs switch generated notes, and the first action checkbox toggles completed state.

### Strong at

Reading comfort, low visual noise, and making the output feel like a saved personal note.

### Weak at

Less efficient for comparing scores or scanning many recommendations quickly.

### Best for

Users who think by writing and want generated project ideas to become a living document.
