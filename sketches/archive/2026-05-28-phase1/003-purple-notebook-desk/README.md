## Variant: Purple Notebook Desk

### Design stance
把灵感池做成长期可用的个人项目书桌：深紫、安静、卡片感、少一点 SaaS 味。

### Key choices
- Layout: 左侧输入笔记，右侧灵感卡片网格。
- Typography: 大标题偏 editorial，卡片标题清晰。
- Color: 深紫纸张表面 + 柔和薰衣草边界。
- Interaction: 生成、收藏、导出都有轻量 toast。

### Trade-offs
- Strong at: 温和、耐看、符合“灵感池”的长期记录属性。
- Weak at: 没有 command-center 那么强的速度和工具感。

### Best for
- 想把产品做成个人灵感工作台，而不是高压仪表盘。
