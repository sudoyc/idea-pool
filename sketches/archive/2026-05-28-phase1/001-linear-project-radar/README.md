## Variant: Linear Project Radar

### Design stance
把灵感池当成精密的项目雷达：深紫暗色、低噪音、强调排序与聚焦。

### Key choices
- Layout: 三栏，左输入、中列表、右详情。
- Typography: 大标题强压缩，正文低亮度。
- Color: 近黑背景 + 紫色/粉紫作为唯一高饱和强调。
- Interaction: 列表点击切换右侧详情，toast 反馈。

### Trade-offs
- Strong at: 高级感、效率感、项目优先级清晰。
- Weak at: 比较冷，不够“灵感笔记”的温和感。

### Best for
- 想把这个工具做成 daily launcher / command center 的方向。
