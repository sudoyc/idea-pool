## Variant: Focus Deck

### Design stance

Decision support. The UI shows one idea at a time so the user can commit, reject, or move on.

### Key layout choices

Centered viewport, collapsed context pill, one large editorial card, subtle stack offset, and three bottom actions.

### Typography choices

Large 36-56px title, 17px summary, tiny wide-tracked metadata, and restrained section labels.

### Interactions

Next/reject advances the card, pin shows feedback, and Edit brief lets the user revise the collapsed prompt sentence.

### Strong at

Reducing comparison anxiety, making one recommendation feel important, and avoiding dense dashboard browsing.

### Weak at

Poor for comparing many ideas side by side or maintaining a historical library.

### Best for

The “I have two hours tonight, choose for me” use case.
