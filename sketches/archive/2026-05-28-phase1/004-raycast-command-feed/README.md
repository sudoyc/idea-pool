## Variant: Raycast Command Feed

### Design stance

Command-first personal launcher. The product feels like invoking a focused tool rather than browsing a dashboard.

### Key layout choices

Single centered column, sticky command bar, filter chips, and a vertical accordion feed. Rows carry only title, one-line rationale, and score until expanded.

### Typography choices

Large command input at 23px, compact semibold row titles, readable 14px detail text, and monospace only for scores/tags.

### Interactions

Click rows to expand/collapse details, filter by idea type, and press Generate to refresh the feed with a toast.

### Strong at

Fast scanning, low chrome, keyboard-command mental model, and avoiding card-grid dashboard patterns.

### Weak at

Less suited to long-form reading or maintaining a permanent library of past generations.

### Best for

Users who want quick idea retrieval and a Raycast-like “tell me what to build now” workflow.
