## Variant: Masonry Pool Wall

### Design stance
主视图像一面私人 idea wall：仍然是许多卡片，但通过不同高度形成空间记忆，而不是用状态栏位分隔。

### Key choices
- Layout: Masonry/瀑布流卡片墙。
- Typography: 摘要保留更多，便于凭空间位置回忆内容。
- Color: 保持克制，卡片本身成为视觉主体。
- Interaction: 左侧 lens 调整可见集合；拖拽时右侧分类窗口滑出。

### Trade-offs
- Strong at: 最接近 “spatial idea pool”。
- Weak at: 真正实现 drag/drop 时 masonry 重排可能更复杂。

### Best for
如果你想让主视图更像个人外脑，而不是任务管理 board。
